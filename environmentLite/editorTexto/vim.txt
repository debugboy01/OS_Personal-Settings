**********VIM

sudo apt-get install vim

editor de texto consola, similar a nano pero con mas funciones.

editar en /etc/vim/vimrc

anadir:

"show number line"
set nu

"tabulation step"
set tabstop=2 softtabstop=2 shiftwidth=2 expandtab

"indent at press enter"
set autoindent

"show current position"
set ruler
